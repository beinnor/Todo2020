User Stories:

Stage 1:

* I can add todos and edit, remove and mark them as completed.
* No authentication or authorization yet.
* Only one list, not multiple lists yet.

Stage 2:

* I can create users, log in and out. And only view own todos.

Stage 3:

* Multiple todolists per user.

* API

Stage 5:
